## Signed Manifest Shape

The run-store writer enriches manifests with:

- `run_id`
- `created_at`
- producer-specific manifest payload fields
- `signature`

The current sample manifest also shows the practical top-level sections operators should expect:

- `schema_version`
- `type`
- `request`
- `reproducibility`
- `model_metadata`
- `execution`
- `signature`

Current signature metadata is generated by `backend/app/signatures.py` and includes:

- `algorithm`
- `signed_at`
- `signature`

`docs/examples/sample_manifest.json` shows the current signed shape using `HMAC-SHA256`.

## Runtime Output Locations

Successful run-producing flows write:

- manifest: `backend/out/manifests/<run_id>.json`
- scenario manifest: `backend/out/scenario_manifests/<run_id>.json`
- artifact directory: `backend/out/artifacts/<run_id>/`
- provenance stream: `backend/out/provenance/<run_id>.jsonl`

## Retrieval Flow

1. submit a run-producing request such as `POST /batch/pareto`
2. extract `run_id` from the response
3. fetch signed metadata:
   - `GET /runs/{run_id}/manifest`
   - `GET /runs/{run_id}/scenario-manifest`
4. fetch detached signature views:
   - `GET /runs/{run_id}/signature`
   - `GET /runs/{run_id}/scenario-signature`
5. enumerate and download artifacts:
   - `GET /runs/{run_id}/artifacts`
   - `GET /runs/{run_id}/artifacts/{artifact_name}`

## Example Commands

```powershell
$body = Get-Content docs/examples/sample_batch_request.json -Raw
$resp = Invoke-RestMethod -Uri "http://localhost:8000/batch/pareto" -Method Post -ContentType "application/json" -Body $body
$runId = $resp.run_id

Invoke-RestMethod -Uri "http://localhost:8000/runs/$runId/manifest"
Invoke-RestMethod -Uri "http://localhost:8000/runs/$runId/scenario-manifest"
Invoke-RestMethod -Uri "http://localhost:8000/runs/$runId/signature"
Invoke-RestMethod -Uri "http://localhost:8000/runs/$runId/scenario-signature"
Invoke-RestMethod -Uri "http://localhost:8000/runs/$runId/artifacts"
Invoke-WebRequest -Uri "http://localhost:8000/runs/$runId/artifacts/evaluation_manifest.json" -OutFile ".\\$runId-evaluation_manifest.json"
Invoke-WebRequest -Uri "http://localhost:8000/runs/$runId/artifacts/thesis_report.md" -OutFile ".\\$runId-thesis_report.md"
```
